# UK data sources

Phase 1.1 of HomeSite ships two UK regions: **Greater London** and the
**West Midlands** conurbation (Birmingham + Wolverhampton + Walsall +
Dudley + Sandwell + Solihull + Coventry).

This document maps each criterion in the catalog to the upstream data
source we plan to ingest in Phase 2. Phase 1.1 ships **synthetic but
spatially coherent** stand-ins generated by per-region seed scripts under
`scripts/`. The synthetic generators encode known geographic priors —
e.g. the Thames flood corridor, M6/M5/M42 noise corridors, prime-area
price bumps, Solihull / Sutton Coldfield affluence patterns.

## Catalog → source mapping

| Criterion | Phase 2 source | Licence | Notes |
|---|---|---|---|
| Flood risk | Environment Agency Flood Map for Planning (England); NRW Flood Map (Wales); SEPA Flood Maps (Scotland) | Open Government Licence v3 | Banded VERY_LOW / LOW / MEDIUM / HIGH. Used as a *categorical* criterion with optional hard-constraint on HIGH. |
| Crime rate | data.police.uk | OGL v3 | Trailing 12 months, all categories. Normalised per 1,000 residents using ONS mid-year estimates. |
| A&E drive time | NHS England estate (A&E sites) + OSM via OSRM/Valhalla | OGL v3 / ODbL | Door-to-door drive time at 08:30 weekday from cell centroid. |
| Fire & rescue response | Home Office FRS incident response statistics + station locations | OGL v3 | Indicative; aggregated from incident-level response times. |
| Primary school rating | Ofsted ratings dataset | OGL v3 | Most recent overall effectiveness for the catchment school. Catchments approximated by nearest-school in Phase 2 v1. |
| Secondary school rating | Ofsted ratings dataset | OGL v3 | As above. |
| GP surgery (walk) | NHS Digital GP Practice register + OSM walk network | OGL v3 / ODbL | Walking time at 4.5 km/h. |

## Listings (deferred)

The UK has no MLS equivalent. Residential listings live behind Rightmove,
Zoopla, OnTheMarket, and a long tail of agent feeds. There is no free,
licensed national feed; commercial agreements per portal are the only
path. Phase 1 ships synthetic listings generated by
`scripts/generate-london-listings.ts` that match Rightmove's field set
(tenure, EPC band, council tax band, square feet, days on market).

Phase 2 options, in rough order of feasibility:
- **Rightmove** — partner / DataLoader; large historical reach.
- **Zoopla** — Member API for portal members; broader EPC + tenure detail.
- **OnTheMarket** — smaller, but more open-feed approach historically.
- **Agent CRM feeds (Reapit, AgentBox, alto)** — bypass portals entirely for
  agent customers, but coverage is fragmented.

We do not intend to scrape the portals — both licensing and TOS are clear.

## Fair-housing / equalities guardrails

The UK has no direct equivalent of the US Fair Housing Act, but the
Equalities Act 2010 prohibits discrimination on protected characteristics
and steering buyers via demographic overlays could plausibly be reviewed
under indirect-discrimination tests.

Operating rules in HomeSite:
- Demographic layers (age, ethnicity, country of origin, language) are
  **never** included in the weighted suitability calculation, even as a
  hidden default. They are display-only.
- The hard-constraint editor never exposes demographic categories.
- Methodology notes are surfaced on every layer, with the data source and
  vintage, so users can interpret what they are weighting.
